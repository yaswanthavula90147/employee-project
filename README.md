# Employee Management System

This is a Spring Boot CRUD REST API project.

## 🚀 Technologies Used
- Java
- Spring Boot
- Spring Data JPA
- MySQL
- REST API

## 📌 Features
- Add Employee
- Get Employee
- Update Employee
- Delete Employee

## ▶️ How to Run
1. Clone the repository
2. Open in IDE (Eclipse/IntelliJ)
3. Run EmployeesApplication.java
4. Use Postman to test APIs
